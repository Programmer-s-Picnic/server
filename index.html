What this server hub is for

Use this subdomain as a clean entry point for backend and server-related experiments.

WS

WebSocket testing

Test secure WSS connections, echo servers, client events, open, close, error, and message handling.

API

API demos

Link small REST-style demos, serverless functions, JSON tools, and backend experiments for students.

CF

Cloudflare Workers

Use Workers for serverless JavaScript, WebSocket endpoints, API routing, and lightweight backend logic.

GH

Static frontends

Host beautiful HTML pages on Cloudflare Pages or GitHub Pages and connect them to secure backends.

SEO

SEO-ready pages

Use proper titles, descriptions, canonical URLs, social images, Open Graph tags, and schema markup.

PP

Programmers Picnic

A practical learning space connected to Champak Roy’s coding lessons, tools, and AI-ML classes.

Live WebSocket tester

Recommended hosting plan

                
Use separate hosting for static pages and server logic.

            


            

                

                    
Host this homepage on Cloudflare Pages

                    
Upload this folder directly or connect a GitHub repository. Point server.learnwithchampak.live to
                        the Pages project.

                


                

                    
Host WebSocket logic on Cloudflare Workers

                    
Use a Worker for the actual WSS endpoint. Start with an echo server, then upgrade later to
                        Durable Objects for shared rooms.

                


                

                    
Connect frontend to backend

                    
Use JavaScript on this page to connect to your Worker URL, for example
                        wss://your-worker.your-subdomain.workers.dev.
